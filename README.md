# Mon App Angular - Tutoriel Homes

Ce projet est une application de recherche de logements développée dans le cadre du tutoriel officiel Angular. Il permet de lister des emplacements et de consulter les détails spécifiques de chaque logement via un système de routage dynamique.

## Fonctionnalités implémentées
- Liste dynamique des logements.
- Utilisation des services pour la gestion des données.
- Routage dynamique avec passage de paramètres (ID).
- Respect des conventions de nommage Angular.

## Installation
1. git clone
2. npm install
3. ng serve
